window,screen,element中不易辨别的属性

发表于 2016-09-19   |   分类于 JavaScript   |     |   阅读次数

window

属性 描述 兼容 使用
window.innerWidth/innerHeight 浏览器可见区域的内宽度、高度（不含浏览器的边框，但包含滚动条） ie9/10、chrome、firefox 假设缩放浏览器的宽度为1000px，高度为500px
alert(window.innerWidth);
//chrome/firefox/ie9/10=>1000
//ie6/7/8=>undefined
alert(window.innerHeight);
//chrome/firefox/ie9/10=>500
//ie6/7/8=>undefined
window.outerWidth/outerHeight 浏览器外宽度（包含浏览器的边框，因各个浏览器的边框边一样，得到的值也是不一样的）。 ie9/10、chrome、firefox 假设缩放浏览器的宽度为1000px，高度为500px
alert(window.outerWidth);
// chrome/firefox/ie9/10=>1002
// ie6/7/8=>undefined
alert(window.outerHeight);
// chrome/firefox/ie9/10=>502
// ie6/7/8=>undefined
window.screenLeft/screenTop 浏览器的位移，表示ie浏览器的内边缘距离屏幕边缘的距离,chrome浏览器的外边缘距离屏幕边缘的距离。 ie6/7/8/9/10、chrome。 alert(window.screenLeft);
// ie=>87
// chrome=>86
// firefox=>undefined
alert(window.screenTop);
// ie=>87
// chrome=>86
// firefox=>undefined
window.screenX/screenY 浏览器的位移，表示：ie9/10浏览器的外边缘距离屏幕边缘的距离,chrome浏览器的外边缘距离屏幕边缘的距离。 ie9/10、chrome、firefox alert(window.screenX);
// chrome/firefox=>86
// ie9/10=>79
// ie6/7/8=>undefined
alert(window.screenY);
// chrome/firefox=>86
// ie9/10=>79
// ie6/7/8=>undefined
window.pageXOffset/pageYOffset 表示浏览器X轴（水平）、Y轴（垂直）滚动条的偏移距离。 ie9/10、chrome、firefox document.onclick=function(){
alert(window.pageXOffset);
// chrome=>200
// firefox=>200
// ie9/10=>200
// ie6/7/8=>undefined
alert(window.pageYOffset);
// chrome=>200
// firefox=>200
// ie9/10=>200
// ie6/7/8=>undefined };
window.scrollX/scrollY 表示浏览器X轴（水平）、Y轴（垂直）滚动条的偏移距离。chrome和firefox中window.pageXOffset和window.scrollX是相等的 chrome、firefox document.onclick=function(){
alert(window.pageXOffset);
// chrome=>200
// firefox=>200
// ie9/10=>200
// ie6/7/8=>undefined
alert(window.pageYOffset);
// chrome=>200
// firefox=>200
// ie9/10=>200
// ie6/7/8=>undefined };

screen

属性 描述 兼容 使用
screen.width/height 屏幕的宽度、高度（指的是屏幕的分辨率，单位为像素） ie6/7/8/9/10、chrome、firefox 假设屏幕的分辨率为1440×900
alert(screen.width);
// chrome/firefox/ie6/7/8/9/10=>1440
alert(screen.height);
// chrome/firefox/ie6/7/8/9/10=>900
screen.availWidth/availHeight 屏幕的可用宽度、高度（通常与屏幕的宽度、高度一致） ie6/7/8/9/10、chrome、firefox alert(screen.width);
// chrome/firefox/ie6/7/8/9/10=>1440
alert(screen.height);
// chrome/firefox/ie6/7/8/9/10=>900

element

属性 描述 兼容 使用
elment.clientWidth/clientHeight 有滚动条时：clientWidth=元素左内边距宽度+元素宽度+元素右内边距宽度-元素垂直滚动条宽度
无滚动条时：clientWidth=元素左内边距宽度+元素宽度+元素右内边距宽度		 chrome、firefox、ie6/7/8/9/10
element.clientLeft/clientTop clientLeft为左边框宽度，clientTop为上边框宽度 chrome、firefox、ie6/7/8/9/10 // borderLeftWidth
alert(oDemo.clientLeft);
// =>50
// borderTopWidth
alert(oDemo.clientTop);
// =>50
element.offsetWidth/offsetHeight offsetWidth=元素左边框宽度+元素左内边距宽度+元素宽度+元素右内边距宽度+元素右边框宽度。 chrome、firefox、ie6/7/8/9/10 // borderLeftWidth+paddingLeftWidth+width+paddingRightWidth+borderRightWidth
alert(oDemo.offsetWidth);
// =>50+60+100+60+50=320
// borderTopWidth+paddingLeftWidth+width+paddingRightWidth+borderRightWidth
alert(oDemo.offsetWidth);
// =>50+60+100+60+50=320
element.offsetLeft/offsetTop chrome：offsetLeft=定位祖先左边框宽度+定位祖先元素左内边距宽度+左位移+左外边距宽度
ie6/7/8/9/10、firefox：offsetLeft=定位祖先元素左内边距宽度+左位移+左外边距宽度。		 chrome、firefox、ie6/7/8/9/10
element.scrollWidth/scrollHeight 有滚动条时：
chrome、firefox、ie8/9/10：左内边距宽度+内容宽度。
ie6/7：左内边距宽度+内容宽度+右内边距宽度（是由CSS的BUG引起）。
无滚动条时：
左内边距宽度+宽度+右内边距宽度。		 chrome、firefox、ie8/9/10、ie6/7（半兼容）
element.scrollLeft/scrollTop 获得水平、垂直滚动条的距离。 chrome、firefox、ie6/7/8/9/10 document.onclick=function(){
// 水平滚动条左距离
alert(oDemo.scrollLeft);
// 垂直滚动条上距离
alert(oDemo.scrollTop); }
